# Admin Panel Dashboard

This is an admin panel dashboard built with Reactjs

## Features

✅ Analytics charts
✅ Dnd Kanban board to manage projects,
✅ A complete Scheduler-calendar to schedule events
✅ Table to manage team

### Technologies Used

-- ReactJS
-- Tailwind CSS
-- ChakraUI
-- MaterialUI
-- React chartjs
-- React Big Calendar
-- react-beautiful-dnd
